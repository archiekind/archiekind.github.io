My Projects

Course Work

I've developed various projects during my time at University. A notable example was a C program to convert and compress image files using a custom file format. By using 2D run length encoding and a variety of other optimisations I developed myself, I achieved a compression ratio of 6.03. A second significant project was designing a hardware implementation of the PRESENT encryption algorithm. I extended the project by designing separate combinatorial, sequential and pipelined designs, and did extensive research into efficiency in Verilog earning full marks on the coursework.

Bristol HyPower Rocketry

I'm leading Software team for the Bristol HyPower rocketry for the 2024/25 development cycle. We compete in the European Rocketry Challenge hosted by the Portugese Space Agency with a target Apogee of 3km. This has extended my knowledge of embedded software development specifically with STM32 boards.

Extended Project Qualification

Researched, analysed and produced a 5000 word report on recent developments in language modelling, focusing on the SwiGLU activation, grouped-query attention (GQA) and reinforcement learning from human feedback (RLHF). To collect primary data for my report I built a 25m parameter language model to conduct ablation tests.
